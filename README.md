# analoguserexperience

Static site for analoguserexperience event design studio.

## Features

- 90s monochrome manga visual palette
- Dark theme with near-black background
- Centered logo and brand area
- Interactive tile-based navigation:
  - Projects
  - Instagram
  - Info
  - Contact
  - Imprint
- Keyboard nav support (arrows + enter)

## Local preview

1. Run a local web server from repo root (e.g. `python3 -m http.server 8000`).
2. Open `http://localhost:8000`.

## Vercel deploy

- Push to repo connected to Vercel.
- Vercel auto-detects static HTML and publishes.
